# Network Architecture Diagram

This folder contains the architecture diagram for Project 1.

The lab environment consists of:

- Verizon ONT Router
- FortiGate 60E Firewall
- Cisco Catalyst 9300 Switch
- Dell PowerEdge R630 Server (ESXi)

Traffic flow:

Internet  
→ Verizon Router  
→ FortiGate Firewall  
→ 802.1Q Trunk  
→ Cisco Catalyst Switch  
→ VLAN Segmented Lab Networks
